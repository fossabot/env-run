# Env Run
[![FOSSA Status](https://app.fossa.com/api/projects/git%2Bgithub.com%2FJunzki%2Fenv-run.svg?type=shield)](https://app.fossa.com/projects/git%2Bgithub.com%2FJunzki%2Fenv-run?ref=badge_shield)


Run command(s) with `.env`.

## Installation

```bash
go install github.com/junzki/env-run@latest
```

## Usage
```
env-run [-e .env] [-d workdir/] -- your command with args
```
